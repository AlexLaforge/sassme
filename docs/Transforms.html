Ligten a color

@param color String expects a hexadecimal
@param percentage Integer expects 0-100 

  lighten : (color, percentage) ->
    hsl = color.mod
      l : color.constrain(color.lightness(), percentage, [0, 100], '+')
    color.rgb2hex(color.hsl2rgb(hsl))

Adjust the hue

@param degrees Integer expects 0-360

    hsl = color.mod
      h : color.constrain_degrees(color.hue(), degrees)
    color.rgb2hex(color.hsl2rgb(hsl))
